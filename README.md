# Amber

### Dependencies

Core library:
* OpenGL (GL, GLU)
* GLFW3
* libPNG
* libJPEG

Unit testing:
* Google testing framework (gtest)
* POSIX Threads (pthread)

### Guide

In order to build and test Amber, run the following commands:
```
mkdir build
cd build
cmake ..
make
```

The `make` command should build and run all unit tests.
